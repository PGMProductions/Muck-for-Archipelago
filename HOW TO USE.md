## How to setup the mod
1) Dowload and install r2modman
2) In r2modman select the game muck, create a new profile and install Archipelago_for_Muck
3) Press Start Modded, the game should launch with the mod loaded (if you want to be sure it worked, you can go to the BepInEx console and check if there is a message that says.`Muck for Archipelago Loaded`)

## How to setup the client
1) Get the lastest apworld on the [release page](https://github.com/PGMProductions/Muck-for-Archipelago/releases) (you don't need the .dll file)
2) Launch the Archipelago launcher, use the `Install Apworld` button and select the downloaded apworld (alternatively, you can manually put it in the `custom_worlds` folder)
3) restart the launcher
4) You should now have a client for muck that you can use to connect to games and that will communicate with the mod
